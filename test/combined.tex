% Image Dimensions
% 1. GLOBAL DEFAULTS & KEEP-ASPECT-RATIO
%    Images are configured globally with a default width and maximum height. 
%    Because 'keepaspectratio' is enabled, an image scales up until it 
%    hits the first of these thresholds. 
%    
%    By default, figures scale to: width = \imagewidthfactor\columnwidth.
%    Extremely tall images will be safely capped at \imagemaxheight.
%
% 2. THE \columnwidth ISSUE
%    Because default values are expanded immediately in the preamble, images placed 
%    inside columns, minipages, or boxes will NOT automatically adapt to the 
%    nested container's smaller \columnwidth. They will mistakenly inherit the 
%    original, global page \columnwidth.
%
% *RECOMMENDATIONS:*
% -> Always explicitly set the image width locally using:
%    [width=\imagewidthfactor\columnwidth]
%    This forces LaTeX to recalculate the width based on the current context.
%
% -> Do not specify a local height. The global \imagemaxheight and 
%    'keepaspectratio' settings remain active, so letting the height adjust 
%    automatically ensures correct proportions without overflow.

% Target width factor relative to the current \columnwidth
\newcommand{\imagewidthfactor}{0.5} 
\newcommand{\imagemaxwidthfactor}{1}

% upper bound for image heigths.
\newcommand{\imagemaxheight}{\textheight}
